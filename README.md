# 🧠 Sentiric Dialog Service

Platformun 'Konuşma Beyni'dir. Niyeti anlar, oturum geçmişini yönetir ve LLM Gateway üzerinden metin üretir.

## 🚀 Hızlı Başlangıç
```bash
cargo run --release
```

## 🏛️ Mimari ve Mantık
* **Geliştirici Kuralları:** Gizli [.context.md](.context.md) dosyasını okuyun (AI Ajanları için zorunludur).
* **Anayasal Konum:** [sentiric-spec/spec/services/dialog.spec.yaml](https://github.com/sentiric/sentiric-spec)
