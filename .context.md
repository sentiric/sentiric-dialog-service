# 🧠 AI SYSTEM INSTRUCTIONS FOR DIALOG-SERVICE
1. **Teknoloji:** Rust 1.84+ (Tokio, Tonic, DashMap, Redis, Lapin).
2. **Hibrit Hafıza (Nano-Edge):** Çağrı geçmişi önce L1 Cache'e (`DashMap`), sonra L2 Cache'e (`Redis`) yazılır. Redis kopsa bile servis `panic!` atıp çökmemeli, L1 Cache ile otonom (Ghost Mode) çalışmaya devam etmelidir.
3. **Medya Yasağı:** Ses dosyaları veya RTP akışlarını doğrudan indiremez/işleyemez. Yalnızca Metin (Token) ve JSON akışı üzerinden çalışır.
4. **mTLS & SUTS:** Güvensiz gRPC iletişimi ve JSON harici düz metin log (örn: `println!`) anayasal bir ihlaldir.
